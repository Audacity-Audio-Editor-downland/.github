## What is Audacity?

Audacity is a free, open-source, cross-platform audio software for multi-track recording and editing . It is used by musicians, podcasters, and audio editors worldwide. Audacity supports recording live audio, converting tapes and records, editing WAV, AIFF, FLAC, MP2, MP3, and Ogg Vorbis files, and applying effects like noise reduction, normalization, and pitch shifting.

## Key Features

| Feature | Description |
|---------|-------------|
| **Multi-Track Editing** | Work with multiple audio tracks simultaneously |
| **Recording** | Record live audio from microphone or mixer |
| **Effects** | Reverb, delay, chorus, EQ, compression, and more |
| **Audio Restoration** | Noise reduction, click removal, and repair tools |
| **Format Support** | WAV, AIFF, MP3, FLAC, OGG, M4A, and more |
| **Spectrogram View** | Visualize and analyze audio frequencies |
| **Plugins** | VST, LV2, and Nyquist support |
| **Multi-Platform** | Windows, macOS, Linux |
| **Open Source** | GPLv2 license |
| **Extensible** | Custom scripts and macros |

## Installation Guide

### Prerequisites

- Windows 10/11
- 4 GB RAM recommended
- 2 GB storage

### Option 1: Windows Installer (Recommended)

**Step 1:** Visit the official Audacity website: `https://www.audacityteam.org/`

**Step 2:** Click **Download** > **Download for Windows**

**Step 3:** Run the installer and follow the on-screen instructions

**Step 4:** Launch Audacity from the Start Menu
